# Memory Card Game

## Description
Build a card matching game where players flip cards to find matching pairs, tracking score and number of moves.

## Requirements
- Grid of face-down cards (8 pairs = 16 cards)
- Click to flip a card, revealing its symbol
- When two cards are flipped, check for a match
- Matched pairs stay face-up; non-matches flip back after a delay
- Track and display: moves taken, pairs found, and game complete status
- Reset/new game button to shuffle and restart

## Examples
- Clicking a card flips it to show its emoji/symbol
- Flipping two matching cards keeps them visible
- Flipping two different cards flips them back after 1 second
- Finding all 8 pairs shows a "You Win!" message
