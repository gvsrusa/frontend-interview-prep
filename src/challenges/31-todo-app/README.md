# Todo App

## Description

Build a fully functional Todo application with CRUD operations, filtering, and localStorage persistence.

## Requirements

- Add new todos via a text input and submit button/enter
- Toggle individual todos between completed and active
- Delete individual todos
- Filter todos by All, Active, and Completed
- Persist todos to localStorage so they survive page refresh
- Show the count of remaining active items
- Do not allow adding empty todos

## Examples

1. Type "Buy groceries" and click Add → appears in the list
2. Click the checkbox → item shows as completed (strikethrough)
3. Click "Active" filter → only unchecked items visible
4. Click "Completed" filter → only checked items visible
5. Refresh the page → todos are still there
