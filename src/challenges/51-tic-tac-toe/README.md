# Tic Tac Toe

## Description
Build a two-player Tic Tac Toe game with winner detection and move history (time travel).

## Requirements
- 3×3 game board with clickable cells
- Alternate turns between X and O
- Detect winner (3 in a row/column/diagonal) or draw
- Display current player and game status
- Move history: click any past move to jump back to that state
- Reset button to start a new game

## Examples
- Clicking an empty cell places X or O and switches turns
- Three X's in a row displays "Winner: X"
- All cells filled with no winner shows "Draw"
- Clicking "Move #3" in history reverts the board to that state
