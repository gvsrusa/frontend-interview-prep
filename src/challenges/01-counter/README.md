# Counter

## Description

Build a simple counter component with increment, decrement, and reset functionality. This is a fundamental React state management exercise.

## Requirements

- Display the current count value, starting at 0
- Provide an **Increment** button that increases the count by 1
- Provide a **Decrement** button that decreases the count by 1
- Provide a **Reset** button that sets the count back to 0
- The count should be able to go negative

## Examples

- Initial render: displays "Count: 0"
- Click Increment 3 times → "Count: 3"
- Click Decrement once → "Count: 2"
- Click Reset → "Count: 0"
