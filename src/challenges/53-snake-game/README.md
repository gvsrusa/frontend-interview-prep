# Snake Game

## Description
Build a classic Snake game controlled with arrow keys. The snake grows when it eats food and the game ends on collision.

## Requirements
- Grid-based game board (20×20)
- Snake moves continuously in the current direction
- Arrow keys change direction (prevent 180° reversal)
- Food spawns at random empty positions
- Eating food grows the snake and increases score
- Game over on wall collision or self-collision
- Display score and game over state
- Start/restart button

## Examples
- Snake moves right by default on game start
- Pressing ArrowUp changes direction upward
- Eating food makes the snake one segment longer
- Hitting the wall displays "Game Over" with final score
