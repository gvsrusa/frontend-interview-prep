# File Upload

## Description
Build a drag-and-drop file upload component with simulated progress bars and cancel functionality.

## Requirements
- Drag-and-drop zone that accepts files
- Also support clicking to select files via input
- Show each file with a progress bar during simulated upload
- Cancel button to abort an in-progress upload
- Display file name, size, and upload status
- Support multiple files simultaneously

## Examples
- Dragging files onto the drop zone adds them to the upload queue
- Each file shows a progress bar filling from 0% to 100%
- Clicking cancel stops the upload and marks it as cancelled
- Completed uploads show a success indicator
