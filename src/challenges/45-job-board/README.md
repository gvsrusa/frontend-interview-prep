# Job Board

## Description
Fetch and display job postings with pagination using mock data. Users can navigate between pages of job listings.

## Requirements
- Display job postings with title, company, location, and type
- Implement pagination (Previous/Next buttons)
- Show loading state while fetching
- Handle error states
- Display current page and total pages

## Examples
- Page loads showing 6 job postings per page
- Clicking "Next" fetches and displays the next page
- "Previous" button is disabled on page 1
- "Next" button is disabled on the last page
