# Weather Dashboard

## Description
Build a weather dashboard that searches for cities, displays weather data from a mock API, and maintains a search history.

## Requirements
- Search input for city names
- Fetch weather data from a mock API
- Display temperature, condition, humidity, wind speed
- Show loading and error states
- Maintain recent search history (last 5 cities)
- Clicking a history item re-searches that city

## Examples
- Searching "London" shows temperature, conditions, etc.
- Searching an unknown city shows an error message
- Recent searches appear as clickable chips below the search bar
