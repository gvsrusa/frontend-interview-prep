# GitHub User Search

## Description
Search for GitHub users and display user cards with their repositories using mock data.

## Requirements
- Search input to filter users by name/username
- Display user cards with avatar placeholder, name, bio, and follower count
- Click a user card to expand and show their repositories
- Each repo shows name, description, stars, and language
- Loading and error states
- Show "No users found" when search has no matches

## Examples
- Typing "dan" shows users whose name/username contains "dan"
- Clicking a user card expands to show their top repos
- Searching an unknown name shows "No users found"
