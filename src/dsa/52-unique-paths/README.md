# Unique Paths

## Description

A robot is located at the top-left corner of an `m × n` grid. It can only move either **right** or **down** at any point. Count the number of unique paths to reach the bottom-right corner.

## Examples

```
Input: m = 3, n = 7
Output: 28

Input: m = 3, n = 3
Output: 6
```

## Constraints

- `1 <= m, n <= 100`

## Complexity

- **Time:** O(m × n)
- **Space:** O(n) with space-optimized approach, O(m × n) with full table
