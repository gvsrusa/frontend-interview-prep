# Merge Sort

## Description

Implement the merge sort algorithm. Given an array of numbers, return a new sorted array in ascending order.

## Examples

```
Input: [5, 2, 4, 6, 1, 3]
Output: [1, 2, 3, 4, 5, 6]

Input: [3, 1, 2, 3, 1]
Output: [1, 1, 2, 3, 3]
```

## Constraints

- `0 <= nums.length <= 10^4`
- `-10^5 <= nums[i] <= 10^5`

## Complexity

- **Time:** O(n log n) in all cases
- **Space:** O(n) for the auxiliary arrays during merge
