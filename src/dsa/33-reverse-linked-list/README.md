# Reverse Linked List

## Description

Given the head of a singly linked list, reverse the list iteratively and return the reversed list.

## Examples

```
Input: 1 -> 2 -> 3 -> 4 -> 5
Output: 5 -> 4 -> 3 -> 2 -> 1

Input: 1 -> 2
Output: 2 -> 1

Input: null
Output: null
```

## Constraints

- The number of nodes in the list is in the range `[0, 5000]`.
- `-5000 <= Node.val <= 5000`
- Solve it iteratively (not recursively).

## Complexity

- **Time:** O(n) where n is the number of nodes
- **Space:** O(1) — in-place reversal
